## Project Overview
This project addressed key point matching in argument mining by modeling semantic overlap between arguments and canonical key points.

## What We Built
- Designed feature construction based on part-of-speech-informed decomposition.
- Fine-tuned DeBERTa for overlap scoring and ranking.
- Evaluated using mean average precision on shared-task benchmarks.

## Why It Matters
Reliable key point matching enables scalable summarization and structure extraction for debate and policy corpora.
